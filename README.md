# 2048
Ready to use and **resizable** 2048 game

Hi, BoxBoxJason there,<br/>
I made this pretty easy to use fully resizable 2048 game using JAVA.

### CONFIG:
	By default, the grid is 4x4. To change this go to:
		- src/GameInterface and edit (line 53):
		- GameInterface game = new GameInterface(4); //Change grid size here
	You can chose any number (I would not recommend too high numbers because the display would become too small to see)

### ATTENTION:
The game counts score but there is nothing implemented to save it / compare highest scores.<br/>
There is no "Game Over" message when the game ends.
Feel free to implement those features if you need them

### LAUNCH:
To launch, simply run the default 2048.jar or main in src/GameInterface (that's it)
